Boder Radius

Syntax:

border-radius:

(top-left) (roght-rigt) (bottom-right) (left-left)

Example:

border-radius:

0px 30px 70px 20px

For displaying round corners Border Radius property is used

IE9 and Opera use the same code. Foe Firedox a prefix "-moz-" needs to be added before the code. And for Webkit Browsers "-webkit-" needs to be added before the code.
